Constants

PI

PI (π) is the numerical value defined by IEEE in double precision as 3.14159265358979323846

E

E (e OR Euler's Number) is the numerical value defined by IEEE in double precision as 2.71828182845904523536.

true

true is the boolean value which evaluates to true. It is represented by the integer 1.

false

false is the boolean value which eavlues to false. It is represented by the integer 0.

NaN

NaN is the value which represents non-numbers. It will come out of computations with a 0 denominator or other similar issues. Its numerical value is 0./0.
